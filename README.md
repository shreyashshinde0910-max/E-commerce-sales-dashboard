# E-commerce-sales-dashboard
An interactive Power BI dashboard built to analyze e-commerce sales performance, customer behavior, and product trends.
📊 Mahadev E-Commerce Sales Dashboard

An interactive Power BI dashboard built to analyze sales, profit, and customer performance for an e-commerce business. The report combines KPI cards, trend charts, and category/geography breakdowns to give a quick, at-a-glance view of business health.

Key Features

KPI Cards — Total Sales Amount, Total Quantity Sold, Total Profit, and Average Order Value (AOV)
Profit by Month — column chart tracking profit trends over time
Profit by Sub-Category — bar chart highlighting top and underperforming product sub-categories
Sales by Category & Quantity — donut chart showing quantity distribution across product categories
Payment Mode Analysis — donut chart breaking down order quantity by payment method
Sales by State — bar chart of revenue distribution across customer locations
Top Customers — column chart ranking customers by sales amount
Interactive Slicers — filter the entire report by Quarter and State

Tech Stack

Power BI Desktop (.pbix)
Data model with Orders and Order Details tables, related via relationships
DAX measures for Profit, Quantity, Amount, and AOV
Date hierarchy (Year → Quarter → Month) for time-based analysis

How to Use

Download E-commerce_sales_dashboard.pbix
Open it in Power BI Desktop
Use the Quarter and State slicers to filter insights
Explore profit, sales, and customer trends interactively

Tools used: Power BI Desktop, DAX, Power Query
